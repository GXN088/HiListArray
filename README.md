# HiListArray
В классе Solution публичный метод print(ArrayList<String>) выводит в консоли все элементы списка по порядку.
Сейчас метод реализован с использованием метода списка forEach(), который принимает лямбда-выражение. Необходимо переписать реализацию метода print(ArrayList<String>), чтобы метод списка forEach() вместо лямбда-выражения принимал ссылку на метод.

Метод main() не принимает участие в тестировании.

Вот псевдокод для вашей программы:

```pseudocode
ПРОГРАММА ЗнакомствоСоСсылкамиНаМетоды

    ФУНКЦИЯ main():
        СОЗДАТЬ пустой список strings типа ArrayList<String>
        
        ВЫЗВАТЬ Collections.addAll() для добавления в strings значений:
            "Hello", "Amigo", "JavaRush", "CodeGym"
        
        ВЫЗВАТЬ print(strings)
    КОНЕЦ ФУНКЦИИ

    ФУНКЦИЯ print(принимает список strings типа ArrayList<String>):
        // Использование ссылки на метод для вывода каждого элемента
        ДЛЯ КАЖДОГО элемента В strings ВЫПОЛНИТЬ:
            ВЫВЕСТИ элемент на экран
        // Эквивалентно: strings.forEach(element -> System.out.println(element))
    КОНЕЦ ФУНКЦИИ

КОНЕЦ ПРОГРАММЫ
```

Ключевые моменты в псевдокоде:

1. strings.forEach(System.out::println) - это использование ссылки на метод (method reference) в Java
2. Конструкция означает: "Для каждого элемента списка strings вызови метод println объекта System.out"
3. Это более краткая форма лямбда-выражения: strings.forEach(s -> System.out.println(s))

Пошаговый алгоритм:

```
НАЧАЛО
1. Создать новый пустой список строк
2. Добавить в список 4 строки: "Hello", "Amigo", "JavaRush", "CodeGym"
3. Вызвать функцию print, передав ей этот список
4. В функции print:
   а. Взять первый элемент списка ("Hello")
   б. Вывести его на экран
   в. Взять следующий элемент ("Amigo")
   г. Вывести его на экран
   д. Повторять для всех элементов списка
5. Завершить программу
КОНЕЦ
```

```
Hello
Amigo
JavaRush
CodeGym
```
